Bienvenue sur la partie Git !



Présentation :

Qu'est ce que git ?

Git est un gestionnaire de versions qui permet de stocker des documents et garder une trace de toutes les modifications. Il permet également de faciliter un travail a plusieurs.


Comment l'installer/l'utiliser ?

pour passer par une interface web il y a pour en citer quelques un Github ou BitBicket . Ils vous permettent de faire du versionnage via une interface web.
Il existe également des logiciels avec interface graphique comme TurtleGit.
Vous pouvez aussi utiliser Git en ligne de commande. Sous Windows je vous conseil GitBash.
Sous linix : 

yum install git
apt-get install git




Les commandes utiles :

Initialiser un dépot Git :

git init

Récupérer un dépot Git :

git clone URL_DEPOT

Initialiser son nom et email :

git config --global user.email "VOTRE_EMAIL"
git config --global user.name "PRENOM NOM"

Ajouter vos modifications :

git add .
git commit -am "MESSAGE_DU_COMMIT"
git push

Mettre a jour notre dépot Git :

git pull

Voir les modifications qu'on a effectué depuis le dernier commit :

git status

h4>Pour changer de branche : 
git checkout NOM_BRANCHE_VOULUE

Pour fusionner sa branche avec une autre branche :

git merge AUTRE_BRANCHE
