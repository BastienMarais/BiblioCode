Bienvenue sur la partie LaTeX !



Introduction :

Le LaTeX est un langage informatique compilé qui permet de créer des documents texte le plus professionnellement possible. On peut ajouter que LaTeX permet d'écrire très facilement des formules mathématiques. il est donc trés utilisé dans le monde scientifique.

Installation

Le LaTeX est un langage Compilé il faut donc dans un premier temps installer le compilateur.
Comme compilateur je peux vous conseiller :

  • MIKTEX qui marche sur Windows.
  • MACTEX pour Mac
  • Et KILE pour Linux
  • Des sites en lignes permettent aussi de créer des fichiers LaTeX en drive comme SharLaTeX
Aprés avoir installer le compilateur, je vous conseil d'installer aussi un éditeur LaTeX. Il n'est pas obligatoire mais fortement conseillé.

Explications :

Le code ci-joint est une mise en page type que je vous propose. (libre a vous de la garder ou non).

On peut voir qu'elle se construit en 4 parties :

  • Partie 1 : on appel les Packages qu'on va utiliser (ou non il y a de nombreux Package que j'appel par précaution)
  • Partie 2 : on definie la mise en page avec les espace entre les lignes, les marge , les couleur ect ...
  • Partie 3 : on peut definir ou redefinir des environnement pour les adopter a ce qu'on souhaite.
  • Partie 4 : Dans l'environnement "document" on ecrit notre code LaTeX en lui même.

De base LaTeX ne prend pas en compte l'accentuation. Pour qu'il puisse les prendre en compte il faut écrire la synthase suivante :
\accent{lettre}

Incertion d'une immage:

Exemple :


\begin{center}
	\includegraphics[scale=echelle]{chemin.png}
	\label{nom du label}
	\ref{nom du label} legende de l image.
\end{center}

Explications :

la command \includegraphics permet d'importer une image dans le document. Si vous regardez les package qu'on utilise vous pourrez voir qu'on peut importer d'autre type de fichier que des ".jpg" ou ".png".
"scale" Vous permet de définir l'échelle de votre image, par exemple scale = 0.5 vous donnera une image qui fera la moitié de l'originale. On peut spécifier si on veut changer que la hauteur ou la largeur de l'image.

\label permet de referencier cette image. Cette referencie pourra être reutiliser par la balise \ref
