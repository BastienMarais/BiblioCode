Structure de base et informations utiles :

Code :

<!doctype html> 
<html lang="fr">
	<head>
		<meta charset="utf-8">
		<title> Nom_Onglet </title>
	</head>
	<body>
		Content...
	</body>
</html>

Description :

Tout document HTML doit contenir cette structure afin de respecter les normes W3C. Pour savoir si votre code est en adéquation avec les normes W3C vous pouvez aller sur ce site. Un autre site utile est jsbeautifier, il permet d'uniformiser votre mise en forme sans faire d'erreurs.



Les titres :

Code :

<h1> Mon très grand titre </h1>
<h2> Mon grand titre </h2>
<h3> Mon titre moyen/grand </h3>
<h4> Mon titre moyen </h4>
<h5> Mon petit titre </h5>
<h6> Mon très petit titre </h6>

Résultat :

Mon très grand titre

Mon grand titre

Mon titre moyen/grand

Mon titre moyen

Mon petit titre
Mon très petit titre


Styles pour le texte :

Code :

<i> Je suis en italique </i>
<b> Je suis en gras </b>
<u> Je suis souligné </u>
<q> Je suis une citation </q>
<del> Je suis barré </del>

Résultat :

Je suis un paragraphe


Je suis en italique
